# react-boilerplate

> An opinionated React.js boilerplate

## Tools
* React.js
* React Router
* SASS with Autoprefixer
* AVA Tests
* Istanbul Coverage

## Installation

``` bash
### Install dependencies
yarn

### Start dev server with hot reload
yarn run dev

### Build everything
yarn run build

### Lint files
yarn run lint

### Run the tests
yarn run test

### Check the coverage
yarn run coverage
```
